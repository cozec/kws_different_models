# Keyword Spotting (KWS) — GitHub Projects Reference

Projects for spoken keyword spotting / wake-word detection built on open-source
datasets (mostly **Google Speech Commands v1/v2**). Compiled 2026-07-14.

## Curated lists (start here)
- [zycv/awesome-keyword-spotting](https://github.com/zycv/awesome-keyword-spotting) — definitive curated index of KWS papers, datasets, and code. Best single entry point.
- [keyword-spotting topic](https://github.com/topics/keyword-spotting?l=python) — all repos tagged keyword-spotting (Python).
- [speech-commands-dataset topic](https://github.com/topics/speech-commands-dataset) — repos tagged with the dataset.

## Model collections / frameworks
- [swagshaw/TorchKWS](https://github.com/swagshaw/TorchKWS) — PyTorch implementations of many KWS research papers, trained on Google Speech Commands.
- [hongfeixue/KWS_pytorch](https://github.com/hongfeixue/KWS_pytorch) — DNN, CNN, TDNN, DFSMN, LSTM (ICASSP 2014 small-footprint KWS paper).

## State-of-the-art / transformer & self-supervised
- [phanxuanphucnd/wav2kws](https://github.com/phanxuanphucnd/wav2kws) — Wav2Vec 2.0 based, SOTA on Speech Commands V1 & V2.
- [qute012/Wav2Keyword](https://github.com/qute012/Wav2Keyword) — Wav2Vec 2.0 based, SOTA on V1 & V2.
- [wdjose/keyword-transformer](https://github.com/wdjose/keyword-transformer) — PyTorch reimplementation of "Keyword Transformer" self-attention model.
- [Kirili4ik/kws-attention-pytorch](https://github.com/Kirili4ik/kws-attention-pytorch) — small (~142k param) attention-based model (Shan et al. 2018).

## Beginner-friendly / end-to-end tutorials
- [vineeths96/Spoken-Keyword-Spotting](https://github.com/vineeths96/Spoken-Keyword-Spotting) — **(selected)** CNN + One-Class SVM hybrid; auto-downloads Speech Commands; TensorFlow 2.2 / Python 3.8.
- [Akash100997/Keyword_Spotting](https://github.com/Akash100997/Keyword_Spotting) — Keras/TensorFlow, MFCC features. Simple and readable.
- [NicolaZomer/Keyword_Spotting](https://github.com/NicolaZomer/Keyword_Spotting) — straightforward Speech Commands recognition.
- [ShawnHymel/ei-keyword-spotting](https://github.com/ShawnHymel/ei-keyword-spotting) — embedded / Edge Impulse deployment for on-device KWS.
